# Gestión de Estudiantes (Java - Consola)

## Descripción
Aplicación simple en Java para demostrar:
- Creación de clases y atributos
- Constructor
- Instanciación de objetos
- Uso de métodos

## Cómo ejecutar
1. Abre el proyecto en IntelliJ / Eclipse / NetBeans
2. Ejecuta `App.java`

## Clases
- **Estudiante**: guarda datos (id, nombre, promedio, activo) y tiene métodos para actualizar promedio, dar de baja y mostrar resumen.
- **App**: crea objetos y prueba los métodos.
